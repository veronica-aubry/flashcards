JavaScript
a high-level, dynamic, untyped, and interpreted programming language.
NaN
Not-A-Number. Technically a number. In modern browsers, NaN is a non-configurable, non-writable property.
Variable naming conventions
Camelcase: first word uncapitalized, second word first letter capitalized.
+=
Shorthand for variable = variable + x
Function
Code that performs a task. Has parameters inside of it.
Method
An action called on an object.
Argument to a method
Additional, sometimes optional, information given to a method. In .toFixed(x), x is the argument
String
A string is an arbitrary sequence of characters.
Chain Methods
Chaining methods occurs when you call a method on another method.
Escape
Escaping is used to tell the computer to literally interpret the character. Also called backslash escaping.
Modulo
finds the remainder after division of one number by another
Operators
Arithmetic operators take numerical values (either literals or variables) as their operands and return a single numerical value.
Alert
An Alert is a JavaScript event. When an alert is invoked a popup appears on the browser.
Commenting Code
The act of describing your code in natural language. This description is not interpreted by the computer. "//" is used to comment code in JS.
Boolean
a binary variable, having two possible values called "true" and "false."
Undefined
The undefined property indicates that a variable has not been assigned a value.
Parameter
a numerical or other measurable factor forming one of a set that defines a system or sets the conditions of its operation.
Returned
Returned is used to print value to console
